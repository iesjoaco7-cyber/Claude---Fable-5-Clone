# Collaboration Plan

The goal is to attract 1–2 real external contributors quickly with small, safe, useful tasks.

## Who to invite

Look for people who are interested in:

- Claude Skills.
- Prompt engineering.
- GitHub Actions.
- Technical documentation.
- Python scripting.
- UI/UX review prompts.

## Outreach message

```txt
Hola! Estoy armando un repo open source llamado Claude---Fable-5-Clone. No clona modelos privados: es una Skill/metodología segura para Claude.ai y Claude Code. Ya tiene README, Skill, evals y GitHub Actions. ¿Te gustaría tomar un issue chico marcado como good first issue y aparecer como contribuidor/a en la release v0.1.0?
```

## Contributor onboarding

1. Point them to `CONTRIBUTING.md`.
2. Assign a `good first issue`.
3. Ask for a small PR, not a large rewrite.
4. Review quickly.
5. Thank them in release notes.

## Public trust signals

- At least one merged PR from an external GitHub account.
- Good first issues opened publicly.
- Passing GitHub Actions.
- `CHANGELOG.md` updated.
- Release `v0.1.0` published.
